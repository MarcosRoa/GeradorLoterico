📱 Telas do App

🏠 Home (index.tsx)

Tela inicial com grid de loterias, exibe créditos e status PRO

🎲 Generate (generate.tsx)

Geração de palpites com IA, seleção de quantidade e modos

📜 History (history.tsx)

Histórico de palpites gerados pelo usuário

👤 Profile (profile.tsx)

Perfil do usuário, saldo, compra de créditos e logout

🔐 Login (login.tsx)

Autenticação com email/senha, Google e modo convidado

⚙️ Tecnologias

  • React Native - Framework mobile
  • Expo - Desenvolvimento e build
  • Expo Router - Navegação
  • Zustand - Gerenciamento de estado
  • Firebase Auth - Autenticação
  • Axios - Requisições HTTP

📦 Scripts

npm start      # Inicia o Expo
npm run android # Roda no Android
npm run ios     # Roda no iOS
npm run web     # Roda no navegador
