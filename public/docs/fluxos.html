🔄 Fluxos do Sistema v7.0

1. Fluxo de Login

1 Usuário clica em "Entrar com Google/Facebook/Email"
2 Firebase Auth retorna usuário autenticado com UID
3 Frontend chama GET /api/credits com header X-User-Id
4 API consulta/cria usuário no Supabase
5 API retorna saldo e status PRO
6 Frontend atualiza interface com os dados

2. Fluxo de Geração de Palpites

1 Usuário configura parâmetros (quantidade, modo, bolão)
2 Frontend chama POST /api/generate com os parâmetros
3 API valida saldo do usuário no Supabase
4 API gera números (IA ou aleatório)
5 API debita créditos e salva no histórico
6 Frontend exibe jogos e atualiza saldo

3. Fluxo de Compra de Créditos

1 Usuário escolhe valor e clica em comprar
2 Frontend chama POST /api/payments/create
3 API adiciona créditos (modo simulação)
4 Frontend atualiza saldo exibido

4. Fluxo de Treinamento da IA

1 Usuário clica em "Treinar IA"
2 Frontend carrega CSV da loteria atual
3 Calcula frequência e atraso dos números
4 Gera score e treina o modelo
5 IA fica pronta para gerar palpites
