🧠 Loterias V.7.0 JS

Documentação dos módulos JavaScript

📘 Módulos do Sistema (v7.0)

Frontend JavaScript (arquivos em public/js/)

🔌 api-client.js
Cliente HTTP para comunicação com a API
Funções:
• getCredits() - Busca saldo do usuário
• getProStatus() - Busca status PRO
• generateGames() - Gera jogos via API
• createPayment() - Compra créditos
🔐 auth.js
Autenticação Firebase (Google/Facebook/Email)
Funções:
• loginGoogle() - Login com Google
• loginFacebook() - Login com Facebook
• logout() - Logout do usuário
🔄 auth-handler.js
Processamento pós-login
Funções:
• processarLogin() - Carrega dados do usuário
• Atualiza créditos e status PRO
🎲 jogos.js
Geração de palpites
Funções:
• gerarJogos() - Gera jogos via IA ou aleatório
• validarSaldoEAcesso() - Valida créditos
🎨 user-interface.js
Interface do usuário
Funções:
• atualizarInterfaceUsuario() - Renderiza UI
• comprarCreditos() - Compra créditos
• abrirPerfil() - Abre perfil
📊 loterias.js
Gerenciamento de loterias
Funções:
• selecionarLoteria() - Troca loteria
• carregarCSV() - Carrega dados históricos
• filtrarDados() - Filtra por período
🧠 ia.js
Inteligência Artificial (frontend)
Funções:
• AdvancedLotteryAI - Classe de IA
• predizerIAEspecialista() - Gera palpites
📈 ia-training.js
Treinamento da IA
Funções:
• treinarIAComFiltrosAtuais() - Treina modelo
• executarBacktesting() - Testa precisão
🛠️ utils.js
Utilitários gerais
Funções:
• mostrarToast() - Notificações
• toggleTema() - Alterna tema claro/escuro
• getModoTexto() - Traduz modos de IA
🌍 globals.js
Variáveis globais
Variáveis:
• isUserPro, creditosUsuario, usuarioAtual
🚀 init.js
Inicialização do sistema
Funções:
• init() - Carrega grid e autenticação
📋 estatisticas.js
Cálculos estatísticos
Funções:
• calcularFrequenciaNumeros()
• calcularDuplasMaisSorteadas()
• calcularTriplasMaisSorteadas()

📦 Dependências

- Firebase Auth (Google/Facebook/Email)
- Supabase (via API, não direto)
- TensorFlow.js (IA)
- jsPDF (exportação PDF)
