🧠 Loterias V.7.0 API

Documentação da API REST

🗄️ API - Endpoints

API centralizada via Vercel Functions com TypeScript

📡 Endpoints Disponíveis

GET /api/credits
Retorna o saldo do usuário logado
Headers: X-User-Id (UID do Firebase)
Response: { success: true, credits: number, isPro: boolean }
GET /api/pro/status
Retorna status do plano PRO
Response: { success: true, isPro: boolean, daysLeft: number }
POST /api/generate
Gera jogos e debita créditos
Body: { uid, lottery, quantity, mode, extraNumbers }
Response: { success, games, creditsSpent, creditsRemaining }
POST /api/payments/create
Cria cobrança PIX (simulação)
Body: { userId, amount }
Response: { success, mode: 'simulation', newBalance }
GET /api/health
Health check para monitoramento
Response: { status, timestamp, database }

🔐 Autenticação

As requisições (exceto webhook) requerem o header:

X-User-Id: {firebase_uid}
